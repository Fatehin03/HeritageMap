HeritageMap

Welcome to the Geospatial Explorer. This project visualizes the historical geography and cultural landscape of Palestine within a global context.

Geospatial Palestine

Interactive Historical Boundary Survey

Global Context

The map provides full global navigation while maintaining a primary focus on the historical region of Palestine.

Cartography

Boundaries are rendered using high-fidelity GeoJSON vectors to ensure clarity across all zoom levels.

Tech Stack

Built with Leaflet.js, CartoDB tiling, and Tailwind CSS for a professional, scalable web experience.
